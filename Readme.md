# Mail server

Simple mail server for easiest mailing

## Getting Started
```
# mkdir -p /data/docker/mailer/application
# git clone git@github.com:Vasary/709b13.git /data/docker/mailer/application
```

### Starting
```
# cd /data/docker/mailer/application
# sudo make build
# sudo make start
```

## Allowed
* 25
* 110
* 143
